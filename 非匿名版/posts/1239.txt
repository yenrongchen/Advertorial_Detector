標題：#國外遊記 大陸旅遊

華東7日6夜 22人團 完整版行程報價單

出發：2026/07/14（一）高雄→上海
返程：2026/07/20（日）上海→高雄
天數：7天6夜
人數：22人（10間房，雙人房）
航班：早去晚回（華航/華信）
交通：全程45座大三通巴士（3年內新車）
住宿：全程近3年新開/翻新酒店
景點：全含門票＋宋城千古情＋吳宮御宴＋西柵門票
一、每日行程（最終優化版）

Day 1｜07/14 高雄 ✈ 上海

• 高雄小港機場集合，搭早班機飛上海

• 專車接機 → 午餐

• 下午：【城隍廟商圈】

• 晚上：外灘夜景、南京路

• 住宿：上海國際旅遊度假區鉑萃酒店 或 同級新酒店

Day 2｜07/15 上海市區

• 上午：南京路步行街 → 外灘萬國建築群

• 下午：新天地 / 田子坊

• 傍晚：外灘日夜景一次拍

• 晚上：【黃浦江遊船】

• 住宿：同前

Day 3｜07/16 上海 → 蘇州

• 上午：蘇州博物館

• 下午：平江路歷史街區 → 山塘街搖櫓船

• 晚餐：【吳宮御宴】（含換裝體驗）

• 住宿：蘇州古運河逸衡酒店 / 萬楓 或 同級新酒店

Day 4｜07/17 蘇州 → 湖州

• 上午：獅子林 → 寒山寺

• 下午：南太湖漁人碼頭

• 晚上：【醉美太湖歌舞秀＋水舞秀】

• 住宿：湖州龍之夢歡樂世界大酒店 或 同級新酒店

Day 5｜07/18 湖州 → 杭州

• 上午：雷峰塔

• 下午：【宋城主題樂園＋宋城千古情表演】

• 晚餐：杭州風味（東坡肉、西湖醋魚）

• 住宿：杭州臨平皇冠假日 / 富陽東方茂 或 同級新酒店

Day 6｜07/19 杭州 → 烏鎮（西柵內）

• 上午：西湖遊船 → 三潭印月

• 下午：前往烏鎮西柵，先入住再遊覽

• 晚上：西柵夜景自由漫步

• 住宿：烏鎮堤上度假酒店 / 枕水度假酒店（西柵內最新）

Day 7｜07/20 烏鎮 → 上海野生動物園 ✈ 高雄

• 上午：烏鎮晨景

• 下午：【上海野生動物園】

• 專車送至浦東機場，搭晚班機返高雄

• 賦歸
二、住宿標準（全程新酒店）

1. 上海：國際旅遊度假區鉑萃酒店 或 同級（近3年新）

2. 蘇州：古運河逸衡 / 萬楓 或 同級

3. 湖州：龍之夢歡樂世界大酒店 或 同級

4. 杭州：臨平皇冠假日 / 富陽東方茂 或 同級

5. 烏鎮：西柵內堤上度假酒店 / 枕水（含西柵門票）
共 6晚 / 10間房（有兩位小孩可以加床，如果不能加床是退費還是要訂滿11間房呢？）

• 全程：45座大三通巴士

• 含：司機、油資、過路費、停車費、司機食宿
